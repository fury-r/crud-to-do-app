## Frontend
``
cd app
``

### install packages
``
npm i
``

### start
``
npm run dev
``

## Backend
``
cd backend
``

### install packages
``

npm i
``

### only do this if db is not there under src/db/
``
ts-node ./src/db/init_db.ts
``
### start
``
npm start
``
